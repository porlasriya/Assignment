# Beyond Simple Segmentation: K-Means Customer Analysis

##  Overview
This project explores customer segmentation using K-Means clustering, enhanced with validation techniques and multi-dimensional analysis.

Unlike basic clustering implementations, this work focuses on:
- Identifying optimal cluster numbers using validation methods
- Interpreting customer segments in a business context
- Extending clustering to higher dimensions for deeper insights

---

##  Objectives
- Segment customers based on income and spending behaviour
- Determine optimal clusters using Elbow Method and Silhouette Score
- Analyse customer groups using statistical profiling
- Extend analysis to 3D clustering using demographic features

---

##  Dataset
- Mall Customers dataset
- Features used:
  - Age
  - Annual Income (k$)
  - Spending Score (1–100)

---

##  Methods Used

### K-Means Clustering
Groups customers into distinct segments based on similarity.

###  Elbow Method
Identifies optimal number of clusters using WCSS.

###  Silhouette Score
Validates cluster separation and cohesion.
